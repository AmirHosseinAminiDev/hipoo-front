# وب‌سایت باشگاه آرتمیس

پروژه چندصفحه‌ای React + Vite + Tailwind CSS برای باشگاه بدنسازی آرتمیس با رابط کاربری راست‌چین و متن فارسی.

## شروع سریع

```bash
npm install
npm run dev
```

سایت شامل صفحات خانه، خدمات، پلن‌ها، مربیان (با صفحه جزئیات)، کلاس‌ها، درباره ما و تماس است. طراحی کاملاً Dark و RTL با فونت فارسی Vazirmatn پیاده‌سازی شده است.
